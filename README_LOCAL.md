# GoPark 로컬 실행 가이드

이 프로젝트는 Vite를 기반으로 하며, GNU Go AI 엔진(WASM)을 사용하여 동작합니다.

## 실행 방법

1.  **배치 파일 실행**:
    *   프로젝트 루트 폴더에 있는 `run_gopark.bat` 파일을 더블 클릭합니다.
    *   최초 실행 시 필요한 라이브러리(`node_modules`)를 자동으로 설치합니다.
    *   설치가 완료되면 개발 서버가 가동되고 브라우저가 열립니다.

2.  **수동 실행 (터미널)**:
    ```bash
    npm install
    npm run dev
    ```

## 주의 사항
*   **Node.js**: 시스템에 Node.js가 설치되어 있어야 합니다. (LTS 버전 권장)
*   **브라우저**: 최신 버전의 Chrome, Edge, Safari 등 Web Worker와 WASM을 지원하는 브라우저에서 최적으로 동작합니다.

## 주요 기능
*   **Tutor NEXUS**: 실시간 AI 대국 및 코칭 메시지 제공.
*   **백그라운드 AI**: Web Worker를 사용하여 AI 수읽기 중에도 화면이 멈추지 않습니다.
*   **자동 동기화**: 엔진의 실제 돌 상태를 실시간으로 동기화하여 정확한 대국이 가능합니다.
